1. Descriptive Statistics

These are your go-to tools for summarizing data without panicking:

  • Mean: The average (add 'em up, divide by how many)
  • Median: The middle value (good when outliers mess up your mean)
  • Mode: The most popular number (literally)

Formula for Mean:

μ = (Σx) / n

2. Variance & Standard Deviation

These tell you how spread out your data is. If your results are all over the place, these guys will snitch.

  • Variance: Average squared difference from the mean
  • Standard Deviation: The square root of variance (aka more readable)

Formulas:

σ² = (Σ(x - μ)²) / n
σ = √σ²

3. Probability Distributions

These show how likely different outcomes are. Think of them like data mood rings.

  • Normal Distribution: The classic bell curve
  • Binomial Distribution: Great for yes/no data (e.g., coin flips)
  • Poisson Distribution: Counts how often something happens in a fixed time/space

The Normal Distribution formula:

f(x) = (1 / √(2πσ²)) * e-(x-μ)² / 2σ²

4. t-tests

Want to know if two groups are *really* different or just being dramatic? Use a t-test.

Formula for independent samples t-test:

t = (X̄₁ - X̄₂) / √((s₁²/n₁) + (s₂²/n₂))

5. ANOVA (Analysis of Variance)

Like a t-test, but for three or more groups. Because scientists love comparing everything.

F-ratio formula:

F = variance between groups / variance within groups

6. Chi-Square Tests

Use this when your data is in categories, and you want to know if the results are random or actually say something.

Formula:

χ² = Σ((O - E)² / E)
Where O = observed, E = expected

7. Correlation & Regression

Ever wondered if one variable is dragging another around? These tools help you figure that out.

  • Correlation (r): Measures relationship strength (from -1 to +1)
  • Linear Regression: Predicts values using a straight line — fancy!

Correlation formula:

r = Σ[(x - X̄)(y - Ȳ)] / √Σ(x - X̄)² * Σ(y - Ȳ)²

Regression equation:

Y = a + bX
